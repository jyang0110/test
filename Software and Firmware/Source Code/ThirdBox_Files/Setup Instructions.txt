This sofware must be run on a Raspberry PI 3B device. 

Pre-requirements:
1. Make sure you have newest version of NOOBS installed in your RaspberryPi 3B. You can purchase a pre-installed NOOBS SD card from many retailers, or download NOOBS at the link below: https://www.raspberrypi.org/downloads/noobs/ and follow the software setup guide here: https://www.raspberrypi.org/learning/software-guide/
2. Make sure you install the Raspbian system with NOOBS.
3. Make sure your Raspberry Pi is properly connected to the local network with Ethernet cable.
4. You will need the IP address of your Pi in order to connect the ssh server. Using the ifconfig command will display the IP address.
5. After 2016, Raspbian has the SSH server disabled by default. Make sure to enable the SSH. There are many ways to enable SSH and click the link below if you need any help about SSH setup: https://www.raspberrypi.org/documentation/remote-access/ssh/
6. No libraries need to be installed in this software.
